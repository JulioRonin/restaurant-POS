## 2. ALINEACIÓN ESTRATÉGICA Y TIME-WINDOW

### 2.1 Trimestre Q3 2026 — el trimestre de la verdad

Ambos proyectos operan en la **misma ventana de 90 días** (2-jun → 1-sep-2026), con métricas independientes pero ritmo de gobernanza sincronizado:

```
JUN ──────────► JUL ──────────► AGO ──────────► 1-SEP CIERRE QBR
│ KŌSO          │ KŌSO          │ KŌSO         │
│ Cerrar 3      │ Beta paga     │ Go-Live      │ → 30 clientes
│ blockers      │ 10 clientes   │ 30 clientes  │
│ técnicos      │               │              │
│               │               │              │
│ LORE          │ LORE          │ LORE         │
│ Cerrar #001   │ ~25 títulos   │ ~40 títulos  │ → motor
│ + ops + 12    │ + 1er         │ + PT-BR live │   orgánico
│ títulos       │ tentpole      │ + EN live    │   listo + caja
```

### 2.2 Por qué no hay conflicto operativo

- **Founders distintos** → no compiten por tiempo.
- **Stack tecnológico distinto** → no comparten infraestructura crítica.
- **Mercados distintos** → no canibalizan audiencia.
- **Curvas de capital distintas** → KŌSO necesita capital externo Q3–Q4; Lore es bootstrap.

### 2.3 Lo que SÍ se sincroniza

1. **Entidad legal y fiscal** (si conviene una sola SAS holding con dos líneas, decide PMO en sem 2).
2. **Cuentas bancarias y payout** (Wise/Payoneer pueden ser únicas; banca MXN debe ser separada por proyecto para trazabilidad).
3. **Aprendizajes de marketing** (Meta Ads de KŌSO sirven de práctica para los ads de Lore en M4; CRM/email tools pueden compartir contrato).
4. **Asesoría legal y contable** (1 despacho para ambos = ahorro 30 %).
5. **PMO + asesor + capital pool** (decisiones consolidadas).

## 12. CIERRE EJECUTIVO DEL PORTAFOLIO

El portafolio reúne dos negocios complementarios en perfil de riesgo y curva de monetización. KŌSO POS requiere **velocidad de ejecución comercial** y **capital de crecimiento** (~USD $110K) para capturar el mercado restaurantero mexicano antes de que la competencia local responda. Lore Books requiere **disciplina de cadencia editorial** y **paciencia con el catálogo** para construir un activo compuesto que rinda en el mediano plazo.

La PMO se compromete a:

1. Mantener la **independencia operativa de los dos founders**.
2. Asignar capital en función de **hitos verificables**, no de promesas.
3. Sostener un **estándar de honestidad operativa** en los reportes — cifras inflanas o desviaciones no escaladas son ruptura del contrato de gobierno.
4. Tomar **decisiones de continuidad/pivote/cierre en el QBR Q4 (1-dic-2026)** con la data del primer ciclo completo.
